#### 1.Please install the package(make sure you are in the virtual environment )

```
pip install -r requirements.txt
```

#### 2.Please code it in Terminal to check the database

```
python manage.py makemigrations rango
```

####  3.Please code it in Terminal to update the database

```
python manage.py migrate
```

#### 4.Please code it in Terminal to write the database

```
python populate_rango.py
```
